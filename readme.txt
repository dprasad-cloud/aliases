 Available Shortcuts - Feel free to use flags like -i as in grep command: findpod -i error

         findpod <search_key>        : Find pod across all namespaces
         findsvc <search_key>        : Find service across all namespaces
         finding <search_key>        : Find ingress across all namespaces
         findcm <search_key>         : Find configmap across all namespaces

         desc <search_key>          : Find the pod across all namespaces and get description
         cm <search_key>            : Find the configmap across all namespaces and get properties

         logs <search_key>          : Find the pod across all namespaces and get logs
         logsf <search_key>         : Find the pod across all namespaces and follow logs

         podbash <search_key>       : Find the pod across all namespaces and bash into the pod
         podsh <search_key>         : Find the pod across all namespaces and sh into the pod
         podexec <search_key> <cmd> : Find the pod across all namespaces and run the cmd

         hmvi <search_key>       : Find the pod across all namespaces and run command vi /opt/tomcat/logs/hm.log
         hmcat <search_key>      : Find the pod across all namespaces and run command cat /opt/tomcat/logs/hm.log
         hmtail <search_key>     : Find the pod across all namespaces and run command tail -f /opt/tomcat/logs/hm.log
